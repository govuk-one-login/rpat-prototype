Use this tool to create and manage your service's integration with GOV.UK One Login.

You'll create a service and set up your first integration client. This takes about 10 minutes.

Before you start

You will need:

  • a name for your service
  • a JSON Web Key Set (JWKS) endpoint URL, a public key, or a client secret
